# git-tutorial


Returning to an Old Revision - Dangerous command
```
$ git reset --hard 0ad5a7a6
$ git reset --soft 0ad5a7a6
```

```
$ git push
$ git push --all
$ git push -f
```

```
$ git checkout .
$ git checkout file_name
$ git checkout revistion_number
```

```
$ git clean -f
```
